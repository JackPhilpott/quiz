Mobile App User Guide

Welcome! This is a mobile application that is made to enable you to participate in a geolocation enabled interactive quiz.

Enjoy!!



Key information:

- Please ensure the httpServer.js is running.

- Please ensure all fields within the answer submit are populated.

- Please find more basemap layer selections available by selecting the layers button at the top right of the map





Generating a new question:

1: Firstly, select the flag. Click 'ok' on the alert and you will see your location.

2: Next select the input_antenna. This inserts an interactive layer of questions.

3: Now the user moves to a location where the question marker is within the blue radius circle.

4: Now select tap_and_play. If the marker returns orange then the marker is within the radius, if not the marker remains pink

5: Select the orange marker and a answer form will appear. Fill in answer form and click submit.

- Please ensure all fields in the form and populated, otherwise the answer will not be allowed to submit.

6: Once the answer has been submitted, an alert will indicate if you answered right or wrong. If right then the respective question marker is returned green, if wrong then returned red

7: You can now move on to the next question by following the map to repeat the process above and complete the quiz

Good luck!!
